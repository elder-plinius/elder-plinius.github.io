AutoTemp

AutoTemp is a Python tool and Gradio interface for optimizing language model output quality and reducing hallucinations. It works by running the same prompt at a string of multiple temperature settings for responses and passing the outputs through an eval and scoring step. It gives users more control over hyperparam optimiziation and offers automatic or manual output selection.
